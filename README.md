# Wsx
Websocket framework

# Features
- [x] Router
- [x] Plugin system
- [x] Lifecycle hooks
- [ ] MessagePack compression
- [x] RPC calls
- [x] Pub/Sub
  - [x] Broadcasting via Redis Stack
- [x] End-to-end typescript
- [x] Validation
- [ ] AsyncAPI documentation
